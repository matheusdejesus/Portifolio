A seguir, apresento alguns dos meus trabalhos recentes que demonstram meu progresso e dedicação na área de desenvolvimento web.

Projetos em HTML e CSS

1. Criando um Site de Fotos de Gatos

Tecnologias Utilizadas: HTML

Descrição: Este projeto consiste na criação de um site dedicado a fotos de gatos, com o objetivo de praticar a estruturação de páginas web.

Destaques do Projeto:

  • Organização das imagens em uma galeria.
  • Utilização de tags semânticas para melhorar a acessibilidade

Link do Projeto: https://matheusdejesus.github.io/CatPhotoApp/

2. Criando um Formulário de Registro

Tecnologias Utilizadas: HTML e CSS

Descrição: Desenvolvimento de um formulário de registro para coletar informações de usuários. Este projeto focou na criação de formulários acessíveis e amigáveis ao usuário.

Destaques do Projeto:

  • Campos de entrada para nome, e-mail e senha.
  • Validação básica de formulário utilizando atributos HTML5.
  • Estilização do formulário para uma apresentação clara e limpa.

Link do Projeto: https://matheusdejesus.github.io/Formulario-de-Registro/

3. Formulário de Pesquisa

Tecnologias Utilizadas: HTML e CSS

Descrição: Criação de um formulário de pesquisa para coletar feedback dos usuários sobre diversos tópicos. Este projeto enfatizou a organização de diferentes tipos de campos de entrada.

Destaques do Projeto:

  • Campos de texto, caixas de seleção, botões de rádio e áreas de texto.
  • Layout responsivo para garantir a usabilidade em dispositivos móveis.

Link do Projeto: https://matheusdejesus.github.io/Formulario-de-Pesquisa/

4. Construindo um Menu de Café

Tecnologias Utilizadas: HTML e CSS

Descrição: Desenvolvimento de uma página que apresenta o menu de um café. Este projeto foi criado para praticar o uso de CSS para estilizar listas e itens de menu.

Destaques do Projeto:

  • Listagem de itens do menu com descrições e preços.
  • Estilização com CSS para um design atraente e coerente com a temática do café.
  • Implementação de um layout responsivo.

Link do Projeto: https://matheusdejesus.github.io/Menu-de-Cafe/

5. Design da Página do Programa de Computação da UFOPA

Tecnologias Utilizadas: HTML e CSS

Descrição: Criação de uma página web para o programa de computação da UFOPA. O objetivo foi desenvolver uma interface informativa e atrativa para apresentar informações sobre o programa.

Destaques do Projeto:

  • Estruturação de conteúdo com seções claras e bem definidas.
  • Estilização utilizando CSS para um design moderno.
  • Inclusão de elementos interativos e visuais para melhorar a experiência do usuário.

Link do Projeto: https://matheusdejesus.github.io/Pagina-do-Programa-de-Computacao-da-Ufopa/
